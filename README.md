# Simple-Tic-Tac-Toe

A responsive local-multiplayer Tic Tac Toe app built in TypeScript.

## Run locally

```bash
npm run build
npm run serve
```

Open `http://127.0.0.1:4173` after the server starts.

The app uses Node 24's built-in TypeScript type stripping, so no extra package install is required for the current version.
